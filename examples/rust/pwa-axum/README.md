Neutral TS example with Axum
============================

Example of using Neutral Template System with Web APP and different themes.

Navigate to the examples/rust/pwa-axum directory and then:

```
cargo run --release
```

A server will be available on port 9090

```
http://127.0.0.1:9090/
```
