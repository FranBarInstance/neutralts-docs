§{:replace; … :}

Replace all occurrences in code part.

{:replace; /from/to/ >> code :}

Any delimiter can be used:

{:replace; ~from~to~ >> ... :}
{:replace; ~\~/~ >> ... :}
{:replace; #from#to# >> ... :}
{:replace; |from|to| >> ... :}
{:replace; XfromXtoX >> ... :}
...
