§{:for; … :}

For loop.

{:for; varname from..to >> code :}

{:for; var 1..10 >>
    {:;var:}
:}

§Modifiers:

{:+for; ... :}
{:^for; ... :}

For more details about the “+” modifier see “modifiers”.
